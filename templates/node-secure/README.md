# node-secure

`node-secure` is a minimal TypeScript starter repository for the standard task
contract.

Included:

- `.devcontainer/devcontainer.json`
- `Taskfile.yml`
- `pnpm`-based dependency management
- ESLint, Prettier, Vitest, and TypeScript
- baseline security scanning with `pnpm audit` and `gitleaks`

After opening the template in a devcontainer, run:

```bash
task ci
```
