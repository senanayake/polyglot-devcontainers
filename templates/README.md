# Templates

The repository currently provides two starter templates:

- `python-secure`
- `node-secure`
- `python-node-secure`
- `java-secure`

Each template is designed to satisfy the standard task contract with minimal
customization and to compose local features from `features/`.

Feature building blocks currently include:

- `security-baseline`
- `python-engineering`
- `node-engineering`
- `java-engineering`
- `agent-runtime`

Use templates when you want to start a new repository.

Use [examples](../examples/README.md) when you want a working, documented
workspace that teaches the environments and implemented features first.
