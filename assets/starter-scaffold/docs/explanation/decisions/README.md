# Architecture Decisions

Use this directory for ADRs (architecture decision records).

ADRs capture durable choices such as:

- task contract changes
- framework selections
- major architecture constraints
- security or compliance-driven design decisions

Recommended naming:

- `ADR-0001-short-title.md`
- `ADR-0002-another-decision.md`

Keep ADRs linked to relevant K-Briefs, requirements, and tests so the decision
remains traceable.
