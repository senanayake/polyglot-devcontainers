# Repository Layout

```text
polyglot-devcontainers
├─ .devcontainer/
├─ .github/
├─ docs/
├─ examples/
├─ features/
├─ templates/
├─ README.md
├─ ROADMAP.md
└─ Taskfile.yml
```

## Key directories

- `.devcontainer/`: validated root environment
- `examples/`: concrete example implementations
- `features/`: reusable devcontainer Features
- `templates/`: starter repositories that compose local features
- `docs/`: Diataxis documentation
