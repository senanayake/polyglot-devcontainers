# Features

## security-baseline

Purpose:

- install `task`
- install `gitleaks`

## pandoc

Purpose:

- install `pandoc`
- optionally install XeLaTeX support for PDF generation

## python-engineering

Purpose:

- install `pre-commit`
- install `uv`

## node-engineering

Purpose:

- enable `corepack`
- activate a pinned `pnpm`

## java-engineering

Purpose:

- install a pinned `gradle`
- install a pinned `trivy`
- keep the Java template on a Gradle-first path

## agent-runtime

Purpose:

- install `jq`
- install `ripgrep`
- install `fd-find`
- install `shellcheck`
