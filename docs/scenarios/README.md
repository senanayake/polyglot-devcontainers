## 🎓 About Diátaxis

This documentation follows the [Diátaxis framework](https://diataxis.fr/), which organizes documentation into four types based on user needs:

- **Tutorials** - Learning by doing
- **How-To Guides** - Solving specific problems
- **Reference** - Looking up information
- **Explanation** - Understanding concepts

This structure helps users find the right documentation for their needs quickly and efficiently.
