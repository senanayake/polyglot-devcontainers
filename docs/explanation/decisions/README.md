# Architecture Decisions

Use this directory for durable architecture decision records (ADRs).

ADRs record the choice. K-Briefs record the learning and evidence behind that
choice.

Recommended naming:

- `ADR-0001-short-title.md`
- `ADR-0002-another-decision.md`

Prefer linking ADRs to:

- the K-Briefs that justify the decision
- the requirements or specifications affected by the decision
- the tests or scenarios that verify the decision in practice

When working in this repository, use:

```bash
task init:adr -- "Short decision title"
```
