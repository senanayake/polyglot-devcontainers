# Documentation

This repository uses the Diataxis model to separate learning material from
operational guidance, factual reference, and architectural explanation.

- [Tutorials](./tutorials/README.md): guided learning paths
- [How-to guides](./how-to/README.md): goal-oriented operational instructions
- [Reference](./reference/README.md): factual descriptions of templates,
  features, and contracts
- [Explanation](./explanation/README.md): design rationale and tradeoffs
